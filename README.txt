Проект будет представлять собой приложение с симуляцией физических опытов.
Будет основная вкладка с кнопками для перехода на другие вкладки. Каждая вкладка будет содержать свой физический опыт.
Для каждого опыта будет своя справочная информация, переменные, которые можно настраивать и переменные, которые будут изменяться и 
выводиться по ходу опыта. В любой момент можно поставить опыт на паузу и изучить все данные.
